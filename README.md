# English Vocabulary Learning Prototype
# Overview
This project is a simple web-based prototype designed to help teachers support students who are learning English vocabulary.
Many students struggle with basic vocabulary, especially in early stages of language learning. Teachers also need quick tools to assess a student’s understanding and help them practice important words.

This prototype demonstrates a simple solution: a webpage that presents vocabulary questions and asks students to identify the correct English word. The goal is to help students develop vocabulary while allowing teachers to quickly observe their understanding.
